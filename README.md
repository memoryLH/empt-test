# Empty Test

## 项目描述

Empty Test 是一个基于 HTML5 Canvas 的贪吃蛇小游戏。玩家通过控制蛇的移动方向吃到食物获得分数，游戏会显示当前分数并在本地保存最高分。

## 功能列表

- 开始游戏
- 暂停游戏
- 继续游戏
- 重新开始游戏
- 分数显示
- 最高分本地存储
- 键盘和触摸控制

## 控制说明

### 键盘方向键

使用键盘方向键控制蛇的移动方向：

- `↑`：向上移动
- `↓`：向下移动
- `←`：向左移动
- `→`：向右移动

### 触摸滑动

在触摸设备上，可以在游戏区域内滑动来控制蛇的移动方向：

- 向上滑动：向上移动
- 向下滑动：向下移动
- 向左滑动：向左移动
- 向右滑动：向右移动
